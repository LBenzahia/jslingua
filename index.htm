Arabic javascript library is intended to help web developpers in case they want to design an Arabic related website. You can test all the functionalities that exist in this library.

PS. If you are looking for fancy webpages, this isn't your place. These are testing pages to show what are the scripts about.

These are the testing pages for each

  • ArLang.js:
    This file is used to test Arabic characters, convert between Arabic and Arabic-Indic numerals, and convert Arabic numerals to words (Arabic pronounciation).
  • ArRom.js:
    This file is used to romanitize Arabic letters.
