Arabic javascript library is intended to help web developpers in case they want to design an Japanese related website. You can test all the functionalities that exist in this library.

PS. If you are looking for fancy webpages, this isn't your place. These are testing pages to show what are the scripts about.

These are the testing pages for each script:

  • JaLang.js:
    This file is used to test Japanese characters, convert between hiragana and katakana, and convert Arabic numerals to words (Japanese pronounciation).
  • JaTrans.js:
    This file is used to translaterate Japanese letters.
