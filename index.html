JsLingua

A javascript library to process languages.

Check the code

JsLingua affords these modules:

Lang module:

This module is responsible of languages character detection. It is used to test if a text contains all the characters of a language or it is a hybrid. Also, it affords the function which transforms the numbers to their pronunciation in a certain language.

Test it

Trans module:

It is used for language translateration. Certain languages have many translateration systems.
